
# ItsMyDuty – Service Booking & Repair App

## Overview

**ItsMyDuty**
is an Android application that allows users to search, request, and track repair services such as Mobile, Laptop, and Printer repairs.
The app connects users with service providers and enables real-time service tracking and communication.

---

## Problem Statement

Many users face difficulty in finding reliable repair services quickly. There is no centralized platform to:

* Search nearby repair services
* Request service easily
* Track service status
* Contact technicians directly

---

##  Solution

ItsMyDuty solves this by providing:

*  Service search functionality
*  Easy service request system
*  Order tracking interface
*  Direct call support with technicians
*  Shop-based service listings

---

##  Target Users

* Individuals needing device repair services
* Students & professionals
* Local service providers (shops)

---

##  Features

* User Authentication (Login/Register)
* Search services (Mobile, Laptop, Printer)
* Submit service request with problem description
* View service provider (shop name)
* Track service status (Technician Assigned)
* Call technician directly
* Cancel service request

---

## Tech Stack

### Frontend

* Kotlin
* Jetpack Compose (Modern UI)
* Navigation Component

### Backend

* Firebase Realtime Database / Firestore
* Firebase Authentication

### Tools Used

* Android Studio
* Git & GitHub

---

## App Flow

1. Splash Screen
2. Login / Register
3. Home Screen (Search Services)
4. Select Service
5. Enter Problem Description
6. Request Service
7. Track Order
8. Call Technician / Cancel Request
